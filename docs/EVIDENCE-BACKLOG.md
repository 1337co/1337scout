# Evidence Backlog (On-Grain)

The kit's evidence-recording layer — the compounding record On-Grain prescribes, kept separate from the constitution's law. Each anchored discipline line traces to a recorded justification: the observed failure it prevents, the surface where it happens, the before/after, and the action taken. Volatile facts (justifications, results, element tiers) live here, never in element bodies.

This is **evidence + memory, not authority** — it cites; it does not legislate.

## Evidence schema (per anchored rule)

Each entry records:

- **observed** — the concrete failure seen (not a hypothetical).
- **surface** — where it happens (which element / tool / interaction).
- **before → after** — the change that prevents it.
- **recurrence** — first observation, or the Nth (an axiom-worthy rule needs ≥3 independent recurrences; a safety-critical one earns a rule on a single observation).
- **action** — the rule added or sharpened, and where it lives.

A new axiom or red line requires a recorded ablation result or source record before adoption; an over-cap element requires a justification row above. The constitution carries current law — this record carries the evidence behind it.
